The reminder application allows users to set reminders for their Kin or colleagues to remember tasks and important events. This application will offer a user-friendly interface and users can remind friends or coworkers to do a task at a specific time.

Reminder

  • The ability to set reminders for someone - users will be able to set reminders for friends or colleagues to remind a task at a certain point in time.
  • Ability to view reminders - Once a reminder is set, users will be able to view the reminder in a list.
  • Ability to set up recurring reminders -Users will be able to set up recurring reminders for tasks that need to be completed on a regular basis.
  • Ability to receive notifications - Users will be able to receive notifications for their reminders via push notifications and also send reminders frequently.

Invitation

This application is used to share digital invitation cards with our kin or colleagues. Our app will remind you of that function before a day and in the morning of the day of the event.

  • Invite/Remind your guests : With the Invitation feature, you can quickly create an invitation, add a date and time, upload media, and set the location of the destination to send out reminders to your guest list.

Alaram

A reminder application with innovative features that will help your friend and colleagues stay on top of tasks and to-do lists. The application allows users to set up a reminder alarm that will alert them of a task or event. Users can fill in the details at any time of day, and choose whether the alarm should be one-time, recurring, or daily.

Schedule

This software is very useful for scheduling appointments or meetings and managing daily tasks for others to remind them.

Location
